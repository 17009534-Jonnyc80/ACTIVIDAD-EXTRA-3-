UNIDAD 2

Conceptos básicos para iniciar a programar En esta unidad
aprendí que programar es darle instrucciones claras a la computadora
para que realice una tarea. También comprendí que los programas siguen
una lógica y deben tener un orden. Me pareció importante porque es la
base para empezar a crear cualquier programa.

Tipos de datos y operadores Aprendí que los tipos de datos
permiten guardar diferentes clases de información como números, textos o
valores verdaderos y falsos. Los operadores sirven para hacer
operaciones matemáticas o comparaciones. Este tema es importante porque
permite trabajar con información dentro de un programa.

Estructuras de control condicionales Las estructuras
condicionales permiten que un programa tome decisiones usando
condiciones (por ejemplo, si algo es verdadero o falso). Me pareció
importante porque hace que los programas sean más inteligentes y puedan
reaccionar a diferentes situaciones.

Estructuras de control de bucles Los bucles permiten repetir
una acción varias veces sin escribir el mismo código muchas veces. Son
importantes porque facilitan el trabajo cuando se necesita repetir
procesos.

Conceptos de programación Refuerza la idea de que programar es
resolver problemas mediante pasos ordenados. Destaca la importancia de
la lógica.

Tipos de datos Explica los diferentes tipos de datos y cómo se
utilizan dentro de un programa.

Estructuras de control de flujo explican cómo
funcionan las decisiones y los ciclos dentro de un programa. Me
parecieron importantes porque ayudan a entender cómo controlar el
comportamiento del programa.
